Incident Response — Fundamentos

O que é Resposta a Incidentes
Conjunto de ações tomadas após a identificação de um possível incidente
de segurança para conter, analisar e mitigar impactos.

Fases básicas
1. Identificação
2. Contenção
3. Erradicação
4. Recuperação
5. Lições aprendidas

Exemplo simples
Evento: múltiplas tentativas de login falhas  
Identificação: alerta via log  
Contenção: bloqueio temporário de IP  
Recuperação: redefinição de senha  
Lição aprendida: reforçar política de senha
